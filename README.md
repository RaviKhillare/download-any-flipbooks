# FlipFetch - Premium Flipbook Downloader

A modern, high-performance web tool to download flipbooks from platforms like AnyFlip, FlipHTML5, and PubHTML5 and convert them into high-quality PDFs.

## 🚀 Features
- **Modern UI**: Dark mode with glassmorphism and smooth animations.
- **Auto-Detection**: Recognizes flipbook platforms from the URL.
- **No Backend Required**: Works entirely in the browser using a CORS proxy.
- **Fast PDF Generation**: Uses `jsPDF` for quick conversion.

## 🛠️ Deployment
This project is built with vanilla HTML/CSS/JS and is ready for **GitHub Pages**.

1. Create a repository on GitHub.
2. Push these files to your repository.
3. Enable GitHub Pages in **Settings > Pages**.
